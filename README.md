# kitkat-verilog

This is the VSCode extension for Verilog-HDL/SystemVerilog Syntax Highlighting.

## Features

* Keyword colorlize
* Number colorlize
* Support System Tasks
* Support Directives
* Support some SystemVerilog Features


## Requirements

No special requrement to use this extension.


## Extension Settings

No settings
